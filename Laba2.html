Лабораторна робота №2


Завдання:

Варіант 3. Горизонтальне дворівневе меню, що розкривається при натисканні лівою кнопкою миші. Другий рівень меню приховується, коли вказівник миші залишає область вибраного пункту або меню.



Варіант 10. Організуйте рух зображення з правого верхнього кута вікна до нижнього лівого.

Результат


Код

<!DOCTYPE html>
<html lang="uk">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Рух зображення</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            overflow: hidden;
        }
        #movingImage {
            position: absolute;
            top: 0;
            right: 0;
            width: 100px;
            height: 100px;
            display: none;
        }
    </style>
    <script>
        function startMovingImage() {
            let img = document.getElementById("movingImage");
            img.style.display = "block";
            let posX = window.innerWidth - 100;
            let posY = 0;
            let interval = setInterval(() => {
                if (posX <= 0 && posY >= window.innerHeight - 100) {
                    clearInterval(interval);
                } else {
                    posX -= 5;
                    posY += 5;
                    img.style.left = posX + "px";
                    img.style.top = posY + "px";
                }
            }, 20);
        }
    </script>
</head>
<body>
    <button onclick="startMovingImage()">Результат</button>
    <img id="movingImage" src="image.jpg" alt="Рухоме зображення">
</body>
</html>
